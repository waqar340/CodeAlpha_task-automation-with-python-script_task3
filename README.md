# Task Automation with Python Script

## 📌 Project Overview
Task Automation with Python Script is a simple Python project that automates repetitive tasks such as organizing files, creating folders, renaming files, and generating logs. It helps save time, reduce manual work, and improve productivity.

## ✨ Features
- Organize files based on their extensions.
- Automatically create folders if they do not exist.
- Rename multiple files in bulk.
- Generate a log of completed tasks.
- Easy to modify and extend for other automation needs.

## 🛠️ Technologies Used
- Python 3
- os
- shutil
- pathlib
- datetime
